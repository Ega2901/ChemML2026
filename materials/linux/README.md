# 🐧 Linux: основы работы в терминале

## Навигация по файловой системе

| Команда | Описание | Пример |
|---------|----------|--------|
| `pwd` | Показать текущую директорию | `pwd` → `/home/user/project` |
| `ls` | Список файлов и папок | `ls -la` (подробный список, включая скрытые) |
| `cd` | Перейти в директорию | `cd ~/Documents` |
| `cd ..` | Подняться на уровень выше | `cd ../..` (на два уровня) |
| `cd -` | Вернуться в предыдущую директорию | |
| `tree` | Дерево директорий | `tree -L 2` (глубина 2) |

## Работа с файлами и директориями

| Команда | Описание | Пример |
|---------|----------|--------|
| `mkdir` | Создать директорию | `mkdir -p data/raw` (с родительскими) |
| `touch` | Создать пустой файл | `touch notes.txt` |
| `cp` | Копировать | `cp file.txt backup/` или `cp -r dir/ dir_copy/` |
| `mv` | Переместить / переименовать | `mv old.txt new.txt` |
| `rm` | Удалить файл | `rm file.txt` |
| `rm -r` | Удалить директорию рекурсивно | `rm -r old_dir/` |
| `ln -s` | Создать символическую ссылку | `ln -s /path/to/file link_name` |

> ⚠️ **Осторожно с `rm -rf`** — удаление необратимо! Всегда проверяйте путь перед выполнением.

## Просмотр и редактирование файлов

| Команда | Описание | Пример |
|---------|----------|--------|
| `cat` | Вывести содержимое файла | `cat data.csv` |
| `head` | Первые N строк | `head -20 data.csv` |
| `tail` | Последние N строк | `tail -f log.txt` (следить в реальном времени) |
| `less` | Постраничный просмотр | `less big_file.txt` (выход — `q`) |
| `wc` | Подсчёт строк/слов/символов | `wc -l data.csv` (число строк) |
| `nano` | Простой текстовый редактор | `nano script.py` |
| `vim` | Мощный текстовый редактор | `vim script.py` (выход — `:q!`) |

## Поиск

| Команда | Описание | Пример |
|---------|----------|--------|
| `find` | Поиск файлов | `find . -name "*.py"` |
| `grep` | Поиск текста в файлах | `grep -r "import pandas" *.py` |
| `grep -i` | Поиск без учёта регистра | `grep -i "error" log.txt` |
| `which` | Путь до исполняемого файла | `which python` |

## Перенаправление и конвейеры

| Синтаксис | Описание | Пример |
|-----------|----------|--------|
| `>` | Записать вывод в файл (перезапись) | `echo "hello" > file.txt` |
| `>>` | Дописать в конец файла | `echo "world" >> file.txt` |
| `\|` | Передать вывод одной команды другой | `cat data.csv \| head -5` |
| `2>` | Перенаправить stderr | `python script.py 2> errors.log` |
| `&>` | Перенаправить stdout и stderr | `python script.py &> output.log` |

## Права доступа

| Команда | Описание | Пример |
|---------|----------|--------|
| `chmod` | Изменить права | `chmod +x script.sh` (сделать исполняемым) |
| `chmod 755` | rwxr-xr-x | Владелец — всё, остальные — чтение и запуск |
| `chown` | Изменить владельца | `chown user:group file.txt` |

```
Обозначения прав: r=4 (чтение), w=2 (запись), x=1 (исполнение)
Пример: 644 = rw-r--r-- (владелец читает/пишет, остальные только читают)
```

## Процессы

| Команда | Описание | Пример |
|---------|----------|--------|
| `ps aux` | Список всех процессов | `ps aux \| grep python` |
| `top` / `htop` | Мониторинг процессов | `htop` (интерактивный) |
| `kill` | Завершить процесс | `kill -9 12345` |
| `Ctrl+C` | Прервать текущий процесс | |
| `Ctrl+Z` | Приостановить процесс | `bg` — продолжить в фоне |
| `nohup` | Запуск без привязки к терминалу | `nohup python train.py &` |

## Архивация и сжатие

| Команда | Описание | Пример |
|---------|----------|--------|
| `tar -czf` | Создать архив .tar.gz | `tar -czf archive.tar.gz dir/` |
| `tar -xzf` | Распаковать архив .tar.gz | `tar -xzf archive.tar.gz` |
| `zip` / `unzip` | Работа с .zip | `zip -r archive.zip dir/` |

## Сеть и загрузка файлов

| Команда | Описание | Пример |
|---------|----------|--------|
| `wget` | Скачать файл | `wget https://example.com/data.csv` |
| `curl` | HTTP-запросы | `curl -O https://example.com/file.zip` |
| `scp` | Копирование по SSH | `scp file.txt user@server:/path/` |
| `ssh` | Подключение к серверу | `ssh user@server.com` |

## Переменные окружения и настройка

| Команда | Описание | Пример |
|---------|----------|--------|
| `export` | Установить переменную | `export PATH=$PATH:/new/path` |
| `echo $VAR` | Посмотреть значение | `echo $HOME` |
| `env` | Все переменные окружения | `env \| grep CONDA` |
| `source` | Выполнить скрипт в текущей оболочке | `source ~/.bashrc` |
| `alias` | Создать сокращение | `alias ll='ls -la'` |

## Полезные сочетания клавиш в терминале

| Сочетание | Действие |
|-----------|----------|
| `Tab` | Автодополнение команд и путей |
| `Ctrl+R` | Поиск по истории команд |
| `Ctrl+A` / `Ctrl+E` | Курсор в начало / конец строки |
| `Ctrl+L` | Очистить экран |
| `↑` / `↓` | Навигация по истории команд |
| `!!` | Повторить последнюю команду |
| `!$` | Последний аргумент предыдущей команды |

## 📚 Полезные материалы

### Интерактивные курсы
- [Linux Survival](https://linuxsurvival.com/) — интерактивный туториал для начинающих
- [The Linux Command Line (книга, бесплатная)](https://linuxcommand.org/tlcl.php) — подробный учебник по командной строке
- [Linux Journey](https://linuxjourney.com/) — пошаговое изучение Linux с нуля

### Шпаргалки
- [devhints.io/bash](https://devhints.io/bash) — компактная шпаргалка по Bash
- [cheat.sh](https://cheat.sh/) — справка по командам прямо из терминала: `curl cheat.sh/tar`

### Видео
- [Missing Semester (MIT)](https://missing.csail.mit.edu/) — лекции MIT по инструментам разработки (shell, vim, git и др.)

### Документация
- `man <команда>` — встроенная справка (например, `man grep`)
- `<команда> --help` — краткая справка (например, `ls --help`)
- [explainshell.com](https://explainshell.com/) — разбор любой команды по частям
